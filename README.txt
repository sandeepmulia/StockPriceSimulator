Usage:

1.Load StockPriceSimulator.sln in Visual studio 2015 (and above) and Hit F5/Run
2.In the browser input text field, type NAB.AX to fetch the ticker from the server
3.Click on Refresh prices to start the price update process (see screenshot\filter.jpg)

NOTE: Without typing any symbol, if one were to click on Refresh prices, all the ticker symbols
configured are displayed (see screenshot\AllStocks.jpg)
